## Fonts

Download and place the following fonts here:

### NotoSans (Latin)
- NotoSans-Regular.ttf
- NotoSans-Medium.ttf
- NotoSans-SemiBold.ttf
- NotoSans-Bold.ttf

### NotoSansDevanagari (Sanskrit/Hindi)
- NotoSansDevanagari-Regular.ttf
- NotoSansDevanagari-Medium.ttf

Download from: https://fonts.google.com/noto

These are needed for rendering Sanskrit/Devanagari text in the quotes.
If fonts are not bundled, Flutter will fall back to system fonts.
