## Bundled Bell Sounds

This directory should contain the following audio files (MP3 format):

- `tibetan_bowl.mp3` — Tibetan singing bowl strike
- `soft_chime.mp3` — Soft wind chime
- `singing_bowl.mp3` — Crystal singing bowl
- `temple_bell.mp3` — Temple bell
- `wind_chime.mp3` — Wind chimes
- `zen_bell.mp3` — Zen meditation bell

These should be short (2-5 seconds), high-quality recordings of soothing bell/chime sounds suitable for meditation session signals.

### Sourcing

Use royalty-free meditation bell sounds from:
- freesound.org (CC0 licensed)
- pixabay.com/sound-effects
- Or record your own
